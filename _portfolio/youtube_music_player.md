# YouTubeMusicPlayer

## 프로젝트 소개

**Electron**을 활용하여 만든 데스크탑용 유튜브 뮤직 응용 프로그램입니다. 기존의 웹에서 사용하던 유튜브 뮤직을 응용 프로그램으로 실행시킴으로써 보다 편한 접근성을 제공하고 간단하게 음악을 제어할 수 있는 미니플레이어를 제공합니다.

{% include gallery %}

## 구현 기능

- 미니플레이어를 통한 음악 재생 제어
- 윈도우 썸네일 버튼 패널을 이용한 제어
